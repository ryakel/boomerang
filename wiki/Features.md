# Features

## Core Concept

Every task always comes back. Dismissal is never free — every "not now" requires a "then when."

## Task Management

- **Quick add** — type and hit Enter from the bottom bar
- **Full add modal** — title, notes, due date, labels, Notion link
- **Edit tasks** — tap to expand, or hover for quick action buttons
- **Statuses** — open, snoozed, stale, done
- **Due dates** — with overdue detection and visual indicators
- **Labels** — custom labels with color picker, filterable
- **Extend due dates** — quick +1d / +1w / +2w or custom date

## AI Features (requires Anthropic API key)

- **What Now** — tells you what to work on based on time + energy level
- **Polish** — takes messy brain-dump notes and makes them actionable
- **Date inference** — extracts due dates from natural language ("do this by Friday")
- **Reframe** — when you snooze too many times, AI breaks the task down
- **Smart nudges** — notification messages shaped by your custom instructions

## Snooze System

Four options: Tonight, Tomorrow, This Weekend, Next Week. Each snooze increments a counter. After hitting the reframe threshold (default 3), you must reframe instead of snooze.

## Routines

Recurring tasks (daily/weekly/monthly/quarterly/annually/custom). Routines live in their own screen and automatically spawn task instances when due. Completing the instance logs it on the routine and schedules the next one.

## Notion Integration (requires Notion token)

- Search existing Notion pages and link them to tasks
- AI creates new Notion pages with structured content
- Linked tasks show "Open in Notion" when expanded

## Notifications

Browser push notifications with configurable:
- Frequency (15m / 30m / 1h / 2h)
- Types: overdue tasks, stale tasks, general nudges
- AI-generated nudge messages when custom instructions are set

## Motivational Feedback

- Completion toasts with context-aware messages
- "Done today" counter in header
- Done history with per-task time-on-list tracking
- Reopen tasks with encouraging messages
